** Query
    ~post~ - Get a specific post
    | Arguments | Required | Type | Description            |
    |-----------+----------+------+------------------------|
    | ~postId~  | true     | ~ID~ | Select a specific post |

    ~posts~ - Get all the posts from a given community
    | Arguments     | Required | Type | Description                      |
    |---------------+----------+------+----------------------------------|
    | ~communityId~ | false    | ~ID~ | Select from a specific community |
